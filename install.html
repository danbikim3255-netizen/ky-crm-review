사용법

1

CRM에서 리뷰할 케이스를 엽니다.

2

Branch Office 탭이 보이는 상태에서 "Review" 버튼을 클릭합니다.

3

AI가 Branch Office 내용을 분석하여 한국어 리뷰를 생성합니다. 완료되면 HQ 셀에 자동 입력됩니다.

설정 변경: Review 버튼 옆 ⚙️ 아이콘을 클릭하면 API 키 변경, 첨부파일 자동분석 토글 등을 설정할 수 있습니다.

주요 기능

📝Branch Office 텍스트 + 이미지 자동 추출
📎CRM 첨부파일 분석 (PDF, ZIP, 텍스트)
🔗SharePoint / NAS 링크 자동 탐색
🎬동영상 프레임 추출 분석
🤖Claude AI 한국어 리뷰 생성
💾HQ 셀 자동 입력 (Xrm.Page)

기존 확장 프로그램과의 차이

기존 Chrome 확장 프로그램은 비활성화 또는 제거해 주세요. 동시에 사용하면 버튼이 중복 생성될 수 있습니다.
  • Service Worker 불필요 — 탭 전환 후에도 버튼이 사라지지 않음
  • HQ 셀 쓰기 안정성 향상 — Xrm.Page API 직접 접근
  • 첨부파일 인증 문제 해결 — 페이지의 SharePoint 토큰 공유
  • 동영상 분석 추가 — 메인 페이지에서 직접 프레임 추출 가능

문제 해결

  • 버튼이 안 보여요 — 북마크를 다시 클릭하세요. Branch Office 탭이 열려 있어야 합니다.
  • "스크립트 로딩 실패" — 사내 네트워크(VPN)에 연결되어 있는지 확인하세요.
  • 리뷰가 생성되지 않아요 — ⚙️ 설정에서 API 키가 올바른지 확인하세요.
  • HQ 셀에 입력이 안 돼요 — 케이스 편집 권한이 있는지 확인하세요.
